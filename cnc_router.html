Description

Metal milling CNC router, great for rapid prototyping. Based on my own project and aliexpress parts.

Hardware used:

-generic grbl arduino UNO shield (might upgrade later)

-nema23 stepper motors (DM542 drivers)

-G-PENNY 2.2kW water cooled ER20 spindle

-RPI 3A+ (cncjs server)

Firmware used:

-GRBL 1.1 (opensource, GitHub)

Software used:

-CNCjs (opensource, GitHub)

-FreeCad (CAM, Link)

Other infrastructure:

-cyclone dust separator

-cooling system
