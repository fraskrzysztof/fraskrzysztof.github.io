Description

While printing components for a CNC router project, I discovered that my 3D printer (Ender 3) is slightly skewed in the X and Y axes. Mechanically straightening the printer would be very difficult, so I wrote a calibration script that uses trigonometric functions to correct the G-code script itself. Each layer is shifted based on the Z-axis height and the measured angles of misalignment in the X and Y axes. GUI is designed using TKinter library.
